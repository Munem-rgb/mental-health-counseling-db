# Mental Health Counseling Database (Mini-World)

## Project Overview
This project models a Mental Health Counseling system using a relational database.  
The goal of this project is to simulate a real-world database environment where counselors, patients, and counseling sessions are organized and managed efficiently.

The database was designed using MySQL and focuses on demonstrating database design concepts such as entity relationships, primary keys, foreign keys, and structured data storage.

## Tools Used
- MySQL
- SQL
- Relational Database Design


Relationships were created between tables to ensure proper data organization and integrity.

## Key Concepts Demonstrated
- Relational database design
- Entity-Relationship modeling
- Primary and foreign keys
- Table creation and constraints
- Structured query design

## Example Queries
Some queries used in this project include:
- Retrieving all counseling sessions for a specific patient
- Listing counselors and their assigned patients
- Tracking appointment schedules
- Analyzing session records

## Purpose of the Project
This project was created to practice database modeling and SQL implementation by building a system that reflects a real-world counseling environment.
